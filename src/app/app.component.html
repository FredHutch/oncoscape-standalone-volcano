Differential Expression Analysis

This is a standalone application to explore the results of Differential Expression Analysis. It is an in-progress feature that will soon become available as part of Oncoscape.

Explore existing results

Supported file formats: .csv, .xlsx

Columns: geneID/gene/genes, log2FoldChange/logFC, padj

Example: example.csv

Empty cells: For missing values such as p-values for genes that have a base mean of 0, leave the cell empty.

Run your own Differential Expression Analysis
